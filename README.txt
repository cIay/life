It's Conway's Game of Life.

Compile with 'lein uberjar' or download life-0.1.0-standalone.jar from releases.

Controls:
SPACE - Pause
LEFT ARROW - Reduce the game tick rate
RIGHT ARROW - Increase the game tick rate
ENTER - Clear the screen
MOUSE WHEEL - Zoom
LFET CLICK - Add a cell
RIGHT CLICK - Remove a cell
